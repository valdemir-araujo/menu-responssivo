explicação de flex-box

Flexbox é uma ferramenta do CSS que permite criar layouts responsivos, distribuindo e alinhando elementos em um contêiner. Ele é útil para criar estruturas flexíveis que se adaptam a diferentes tamanhos de tela e dispositivos.

Principais caracteristicas

  • Flexbox é baseado em direções "flex flow"
  • Distribui o espaço entre os itens de forma eficiente
  • Permite ajustar a disposição e o espaçamento dos elementos
  • É excelente para layouts simples e unidimensionais
